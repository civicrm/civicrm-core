## <a name="bugs"></a>Bugs resolved

* **_Composer_: Set version of "brick/money" more precisely ([dev/core#6362](https://lab.civicrm.org/dev/core/-/issues/6362): [#34993](https://github.com/civicrm/civicrm-core/pull/34993))**

  The package "brick/money" had a recent release that will fail to work with current CiviCRM. For composer-based sites (Drupal 9+), this impacts install/upgrade.

* **_Advanced Search_: Fix filtering of custom groups ([#35008](https://github.com/civicrm/civicrm-core/pull/35008))**
* **_Upgrader_: Cannot browse to upgrade-screen on sites with Civi Import ([dev/core#6356](https://lab.civicrm.org/dev/core/-/issues/6356): [#34948](https://github.com/civicrm/civicrm-core/pull/34948))**
