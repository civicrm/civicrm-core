Explanations for some of the file in this directory:

civicrm.mysql  : schema definition file
civicrm_data.mysql: Required data for civicrm
civicrm_generated.mysql: CiviCRM sample data, used for testing. This data file INCLUDES civicrm_data.mysql

civicrm_update_vOLD_vNEW.mysql: db upgrade file from OLD version to NEW version
